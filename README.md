# Betreuungsbüro Maximilian Sporbert

Statische GitHub-Pages-Website des Betreuungsbüros von Maximilian Sporbert.

## Struktur

- `index.html`: Startseite und digitale Kontaktkarte
- `rechtliche-betreuung.html`: Tätigkeitsbereich Rechtliche Betreuung
- `verfahrenspflegschaften.html`: Tätigkeitsbereich Verfahrenspflegschaften
- `kontakt.html`: Kontaktdaten und digitale Kontaktkarte
- `impressum.html`: Platzhalter für das Impressum
- `datenschutz.html`: Platzhalter für die Datenschutzinformationen
- `assets/css/styles.css`: Gestaltung und responsive Darstellung
- `assets/js/availability.js`: Anzeige der telefonischen Erreichbarkeit
- `kontakt.vcf`: Digitale Kontaktkarte

Die Website verwendet keine Frameworks, Build-Tools oder externen Bibliotheken.
